⚙️ Setup Required

This app needs a Firebase project to sync your data across devices. It takes about 5 minutes.

  1. 1Go to console.firebase.google.com and create a new project (name it anchor-tracker)
  2. 2In the project, click Firestore Database → Create database → Start in test mode
  3. 3Go to Authentication → Sign-in method → Enable Google
  4. 4Go to Project Settings (gear icon) → Your apps → Add web app → Copy your firebaseConfig
  5. 5Open index.html in a code editor, find the FIREBASE_CONFIG_HERE section and replace it with your config
  6. 6In Firebase Auth → Settings → Authorised domains → add your GitHub Pages domain (e.g. yourusername.github.io)
  7. 7Push to GitHub Pages and open the live URL

💡 Full step-by-step instructions are also in the README.md file in your repo.
